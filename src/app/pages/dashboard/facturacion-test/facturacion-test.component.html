Instrucciones
¿Cómo usar esta prueba?
  1. Modifica los datos del cliente si lo deseas
  2. Ajusta las cantidades y precios de los productos
  3. Haz clic en "Generar Boleta" o "Generar Factura"
  4. El sistema simulará la emisión del comprobante
  5. Podrás imprimir, descargar PDF o enviar por email
Nota Importante

Esta es una simulación para pruebas. Los comprobantes generados no son válidos para efectos tributarios reales.
